## 🎛️ Using the software

Connect your Teenage Engineering device once the app is open. The software detects your device model automatically. You will see a list of available actions on the main screen. You can select an action to update firmware or manage device settings.

### Updating firmware
Select your device from the menu. Click the Firmware tab. The software compares your current firmware version with the latest version in the directory. If a newer file exists, click Update. The software handles the transfer process. Do not disconnect the USB cable until the progress bar reaches one hundred percent.

### Managing files
You can backup your samples and patches through the Files tab. Choose your device and select the files you want to copy. Click Export to save them to your computer. To add new samples, drag the destination files into the application window and click Import.

## 🛠️ Performing hardware mods

This directory features teardown guides for various units. Read every step before you start. Use the correct tools, such as precision screwdrivers, to avoid damaging the casing. Place your screws in a tray to keep track of them. Modifying hardware carries risks. Proceed only if you feel comfortable working with delicate internal components.

## ❓ Frequently asked questions

**Does this software delete my data?**
The software does not erase internal memory unless you choose to reset the device. We recommend creating a backup of your files before you start any firmware update.

**Is it safe to use custom firmware?**
Custom firmware changes how the device functions. While most mods are stable, firmware changes can affect device performance. Use mods created by trusted community members found in this list.

**Why does Windows block the app?**
Windows protects your computer from unknown software. Because this app comes from an open-source community, Windows does not recognize the publisher. The app is safe to use. Click More info and Run anyway to continue.

**Can I use this on a Mac?**
This version is designed specifically for Windows. Check the repository updates for future compatibility news regarding other operating systems.

**Where do I report issues?**
If you find a broken link or have trouble with a tool, please open an issue on the GitHub repository page. Include your device model and a description of the error so we can help you faster.
